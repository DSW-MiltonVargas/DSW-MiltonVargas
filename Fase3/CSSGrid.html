CSS Grid

Como habiamo señalado Castro González J. el CSS3 se trata de un modelo de diseño, en el cual los elementos se adaptan a un contenedor para ser organizados de forma automática, esto de acuerdo con el tamaño de la pantalla o dispositivo en el que se ejecute.

Allí existe un elemento padre que sostiene a los demás flex por medio de dos ejes “Flex Container”, los elementos hijos se conocen como “Flex Ítem” (Castro González, 2020)(p. 17).
